# Smart Home Automation System

## Overview

The Smart Home Automation System is an IoT-based project designed to automate household devices through sensor monitoring and microcontroller-based control.

The system collects environmental data and automatically controls connected devices based on predefined conditions.

---

## Features

✅ Automatic lighting control  
✅ Environmental monitoring  
✅ Sensor-based automation  
✅ Real-time device response  
✅ Smart device control

---

## Hardware Components

- ESP8266 / ESP32 / Arduino Uno
- Light Sensor (LDR)
- Motion Sensor (PIR)
- Temperature Sensor
- Relay Module
- LED
- Buzzer

---

## Technologies Used

- Arduino IDE
- Embedded C/C++
- IoT Sensors
- Wireless Communication
- Microcontroller Programming

---

## System Workflow

1. Sensors collect environmental information.
2. Microcontroller processes the sensor data.
3. System determines the required action.
4. Connected devices respond automatically.
